## Instrucciones de Ejecución

1.  Clonar el repositorio: git clone https://github.com/KiyoponLZ/Trabajo_integrador_Prog_Senio_NEVADA.git
2.  Abrir el proyecto: Importar la carpeta raíz en tu IDE (IntelliJ IDEA o NetBeans).
3.  Compilar: Debe tener configurado el JDK 21 en la estructura del proyecto.
4.  Ejecutar: Localizar el archivo Main.java dentro del paquete "integrado/prog2/", hacer click derecho y seleccionar --> Run / Ejecutar.
